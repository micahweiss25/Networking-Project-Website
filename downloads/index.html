Test Page

A simple web page for testing the CS484 / CY350X Web Server Programming Project. This page requires three requests from the server. The page includes such interesting HTML elements as:

  • Embedded and inline styles
  • Bold and italic text
  • Unordered list
  • Colored text
  • Colored background
  • A HMMWV ready to drop! (note: HMMWV image is in a subdirectory!)
Even with a single thread and unqueued socket connection requests, this page should serve up fine. It wouldn't hurt to either thread the server or, at at the very least, allow a few queued connections on your server to be sure this page comes up properly.

Images should load without any issue.

Your server should also serve up a video just fine, like this demonstration of sling load operations.

Even JavaScript should work.

Finally, to test your redirect, click for a HMMWV ready to jump.
